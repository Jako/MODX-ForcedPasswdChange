--------------------
Extra: ForcedPasswdChange
--------------------

Version: 1.0.1-pl
 
With the ForcedPasswdChange you can create or update an user and 
activate the forced change of their password in the Manager. When 
checked this option, the user can login but can't do anything 
untill they have changed their password succesful.

see http://rtfm.modx.com/display/ADDON/ForcedPasswdChange
